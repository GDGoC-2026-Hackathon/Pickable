# Pickable

2026년도 GDGoC 해커톤 레포지토리입니다.

## 기술 스택

- **Framework**: Next.js 16.1.6 (App Router)
- **Language**: TypeScript
- **Deployment**: Google Cloud Run
- **Runtime**: Node.js 20

## 시작하기

### 로컬 개발

```bash
# 의존성 설치
npm install

# 개발 서버 실행
npm run dev
```

브라우저에서 [http://localhost:3000](http://localhost:3000)을 열어 확인하세요.

### 빌드

```bash
npm run build
npm start
```
